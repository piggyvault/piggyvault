# Piggy

## Build APK

### Prerequisites

- [OpenJDK 17](https://learn.microsoft.com/en-us/java/openjdk/download#openjdk-17)

```bash
flutter build apk --no-tree-shake-icons
```

Tested on the following environment

- Flutter 3.10.6
- Android SDK version 30.0.3
